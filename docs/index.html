Dictate anywhere

Tap a key, speak, tap again. Raw text lands exactly where your cursor is — in any app, never auto-formatted unless you ask.

Rewrite by voice

Select text, give an instruction — “make this formal”, “turn into bullets”, “translate to Hindi” — and unmute rewrites it in place.

Local-first & private

No unmute account, no server, no telemetry. Audio goes only to Groq via your own API key — or runs fully on-device.

Get started in two minutes

How it works

Two clicks of permission, optional key, then dictate anywhere.

1

Install

Download the DMG, drag unmute to Applications, launch.

2

Grant permissions

Onboarding walks you through Microphone + Accessibility. Optionally paste a free Groq API key for the fastest transcription, or skip to use the on-device model.

3

Just speak

Tap Fn anywhere to dictate, ⇪ to give voice instructions on selected text. That's it.

Privacy

Your words stay yours.

No accounts, no servers, no tracking. Your Groq key is encrypted in the macOS Keychain and never leaves this device. Audio is sent only to Groq via your own key (over TLS) or processed fully on-device.
